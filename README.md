<h2> Functionality </h2>
<p> 
This device can dim the lights according to the position of the potentiometer. It can be also controlled remotely via Wifi with help of mobile app.</p>
<p> The dimmer function is based on the Leading Edge Triac method. The brain of this device is ESP8266 microcontroller, which ensures all timings and connectivity. </p>
